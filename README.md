
# ClipSync 

ClipSync is a full-featured video-sharing platform inspired by YouTube.
Users can upload videos, explore content, interact through comments, like/share/subscribe, and build their own creator identityâ€”all in one seamless experience.

---
## Key Features

- **Upload Videos:** Share your content with the world
- **Like & Dislike:** Engage with videos instantly
- **Comment System:** Add and read comments on any video
- **Subscribe to Channels:** Follow your favorite creators
- **User Authentication:** Secure login/signup
- **Search Videos:** Find content by keywords
- **Responsive UI:** Works across mobile, tablet, and desktop

---


## Tech Stack

- **Frontend:** React
- **State Management:** Redux
- **Backend:** Node.js (Express)
- **Database:** MongoDB
- **Authentication:** JWT 
- **Storage:** Cloudinary
- **Password hashing:** bcryptjs
- **Environment:** dotenv
